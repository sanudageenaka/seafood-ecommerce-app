# Frontend (React + Vite + Tailwind)
## Quickstart
1) Create `.env` with `VITE_API_URL=http://localhost:5000`
2) Install deps: `npm install`
3) Start dev server: `npm run dev`

## Pages
- Home: product grid + search
- Product details
- Login / Register
- Checkout (client-side cart)
- Admin: add/delete products (requires admin token)
